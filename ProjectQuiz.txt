Why should you ensure that the script element that links to the jQuery library file comes before any other script element for a .js file that includes jQuery code?

We need to make sure that the browser downloads the script as soon as possible, while at the same time loading the document. The jQuery library should be the first to load, and will only load once the DOM is ready.

What is a Content Delivery Network (CDN)?  List and explain at least FOUR advantages in using a CDN.

CDNs replicate content from origin servers to servers all over the globe. They significantly reduce load time of information on a page, make traffic more manageable, cause networks to be more secure, and reach mobile customers with more ease.
